# Julia 
## Installing Julia 
When creating the installation directory for Julia, do yourself a favor and change "\...\Julia VERSION" to "\...\Julia-VERSION".
## Adding Julia to PATH in Windows 
- Open the Run program (Win + R), type in "rundll32 sysdm.cpl,EditEnvironmentVariables"
- Highlight Path in User Variables and click EDIT
- Add the following to your PATH list: "C:\Users\USERNAME\AppData\Local\Programs\Julia-VERSION\bin"
